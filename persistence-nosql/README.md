# persistence-nosql

> A Spring Boot module designed to implement a product catalog service using MongoDB and Spring Data MongoDB. Currently under initial development.

## Tech Stack
*   Java 21
*   Spring Boot 3.5.11
*   Gradle 9.x.x
*   MongoDB (Driver managed by Spring Boot)
*   Testcontainers 1.20.4

## Prerequisites
*   Java 21
*   Docker 24.0+ (Required for future integration tests via Testcontainers)

## Getting Started

### Build
```bash
./gradlew :persistence-nosql:build
```
### Run
```bash
./gradlew :persistence-nosql:bootRun
```
